vim 的基本操作 🤸🏼‍♀️

2018-10-02

移动光标

移动到行尾 — $

移动到行首 — 0

移动到段首 — {

移动到段尾 — }

移动到下一个词 — w

移动到上一个词 — b

移动到文档开始 — gg

移动到文档末尾 — G

跳到第n行 — ngg / nG / :n

移动光标到屏幕顶端 — H

移动光标到屏幕中部 — M

移动光标到屏幕底部 — L

移动到上次编辑的位置 — `

编辑操作

后插一行 — o

前插一行 — O

删除一行 — dd

粘贴 — p

交换两行 — ddp

复制 — y

拷贝当前行 — yy / Y

撤销 — u / ctrl + r

缩进所有选择的代码 — >>

反缩进所有选择的代码 — <<
